3. run alembic
    1. mod alembic.ini and alembic/env.py
    2. alembic upgrade head
    3. alembic revision --autogenerate -m "history"
    4. edit alembic history script - sa.Enum(..." lines
        // https://stackoverflow.com/questions/63461381/how-to-use-an-existing-sqlalchemy-enum-in-an-alembic-migration-postgres
        from sqlalchemy.dialects import postgresql
        rename sa.Enum to postgresql.ENUM
        remove enum items and leave name + create_type=False
        ie: postgresql.ENUM(name='ProgramType', create_type=False)
    5. alembic upgrade head
